# Izumi

**Izumi** is a premium Field Force Management application built with Flutter. It tracks work sessions, locations, and visit proofs with a high-end, user-friendly interface.

## Features

- **Authentication**: Secure phone number login with OTP verification.
- **Work Tracking**: 
  - Start/Stop work sessions.
  - Live session timer and distance tracking.
  - Idle/Active state visualization.
- **Visit Proof**: 
  - Integrated camera for capturing visit evidence.
  - Geo-tagged photo previews with metadata (Location, Distributor/Customer type).
- **History & Analytics**:
  - Daily session summaries.
  - Monthly work history.
- **Profile Management**: User details and settings.

## Getting Started

1.  **Prerequisites**: Flutter SDK installed.
2.  **Installation**:
    ```bash
    flutter pub get
    ```
3.  **Run**:
    ```bash
    flutter run
    ```

## Tech Stack

-   **Framework**: Flutter
-   **State Management**: Riverpod
-   **Navigation**: GoRouter
-   **Typography**: Google Fonts (Outfit)
